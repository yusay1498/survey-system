## 📖 用途別ガイド

### 初めてセットアップする場合
1. [開発ガイド](./technical/DEVELOPMENT.md) で環境構築
2. [管理者セットアップガイド](./guides/ADMIN_SETUP.md) で管理者を作成
3. [次のステップ](./guides/NEXT_STEPS.md) でテスト実施

### 結果パターン機能を使う場合
1. [結果パターン設定ガイド](./guides/RESULT_PATTERNS.md) で機能を理解
2. [サンプルデータガイド](./guides/SAMPLE_DATA.md) でサンプルを試す
3. 実際のパターンを作成・テスト

### コードベースを理解したい場合
1. [プロジェクト構造ガイド](./technical/ARCHITECTURE.md) でアーキテクチャを学ぶ
2. [実装内容の説明](./technical/IMPLEMENTATION_SUMMARY.md) で各機能の実装を確認
3. ソースコードを読む
